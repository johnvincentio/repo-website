Page Layout Example No. 1

In this example, you have a three-row table. The height is set in the CSS for the first and second rows. A background image is specified for the second row; the image is 1280 pixels wide and set to no-repeat.

Both the page name (at left) and this block of text are DIVs with CSS-specified position (top and left) and width.
