Page Layout Examples

In these examples, you will see how a large photo image can be used as a cell background for a nice visual effect.

You will also see some useful CSS techniques, expecially for buttons with the hover effect. The CSS is included on each page (because each page is different), and not on an external style sheet.

Warning: The CSS does not always work perfectly in every size browser window. Change the size of the window and see what happens to the page layout!
