Page Layout Example No. 3

In this example, you have a one-row table with two cells. The height of the table is 100%. The width is set in the CSS for the first cell (at left) only. A background image is specified for the first cell; the image is 1280 pixels tall and set to no-repeat.

Both this block of text and the one below it are DIVs with CSS-specified left margins and width. The heading (H1) above also has its width and left margin spec'd in the CSS.
